## DAW Icons

#### Text-based track icons for your DAW.

https://petlyh.github.io/daw-icons/

### Generating icons

- Install the requirements:
    - [Python](https://www.python.org/downloads/)
    - [resvg](https://github.com/RazrFalcon/resvg/releases)
- Run the generator:
```
$ python generate.py
```
